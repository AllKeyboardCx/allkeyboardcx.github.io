# 栈溢出漏洞利用完全指南

## 什么是栈溢出

栈溢出是一种常见的内存安全漏洞，当程序向栈中写入的数据超过了预先分配的缓冲区大小时就会发生。

## 栈的结构

```
高地址
+------------------+
| 函数参数         |
+------------------+
| 返回地址 (EIP)   |  ← 我们要覆盖的目标
+------------------+
| 基址指针 (EBP)   |
+------------------+
| 局部变量         |
+------------------+
低地址
```

## 基本利用方法

### 1. 确定缓冲区大小

通过调试或逆向分析确定缓冲区的精确大小。

### 2. 构造payload

```python
padding = 'A' * 100  # 填充到返回地址
eip = p32(0x08048456)  # shellcode地址
shellcode = b'\x31\xc0\x50\x68\x2f\x2f\x73\x68...'

payload = padding + eip + shellcode
```

### 3. 绕过保护机制

现代系统有多种保护机制：
- ASLR：地址空间布局随机化
- DEP/NX：数据执行保护
- Canary：栈保护

## ROP链构造

当DEP启用时，需要构造ROP链：

```python
rop_chain = p32(write_addr) + p32(next_addr) + p32(fd) + p32(buf) + p32(len)
```
