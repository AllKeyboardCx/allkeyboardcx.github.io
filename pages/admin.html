📖 使用说明

  • 发布文章：填写表单后点击"发布文章"，文章会保存为md文件到对应分类目录。
  • 文件名：建议使用小写字母和连字符，不要包含空格和特殊字符。
  • 标签：多个标签用英文逗号分隔。
  • Markdown：支持标准Markdown语法，包括标题、代码块、列表等。
  • GitHub同步：发布后需要将新文件推送到GitHub，网站才会更新。
  • 本地测试：安装Node.js后运行 npm run build 更新index.json，然后用本地服务器预览。
