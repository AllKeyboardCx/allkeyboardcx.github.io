📚 Linux系统安全笔记

2026-06-10

1. 用户与权限管理

chmod 755 /path/to/file
chown user:group /path/to/file
usermod -aG sudo username

2. 防火墙配置

  • iptables：传统防火墙工具
  • ufw：简化版防火墙配置
  • firewalld：动态防火墙管理

3. 日志监控

重要日志文件位置：

  • /var/log/auth.log - 认证日志
  • /var/log/syslog - 系统日志
  • /var/log/secure - 安全相关日志

🐍 Python安全编程笔记

2026-05-25

1. 代码审计要点

  • SQL注入：使用参数化查询
  • 命令注入：避免使用subprocess
  • 反序列化：谨慎处理pickle数据

2. 安全库推荐

  • cryptography：加密库
  • requests：安全HTTP客户端
  • bandit：代码安全扫描

3. 示例代码

import hashlib
from cryptography.fernet import Fernet

# 密码哈希
hashlib.sha256(password.encode()).hexdigest()

# 对称加密
key = Fernet.generate_key()
cipher = Fernet(key)
encrypted = cipher.encrypt(b"secret data")

📖 密码学基础笔记

2026-04-18

1. 对称加密

  • AES：高级加密标准
  • DES：数据加密标准（已过时）
  • 3DES：三重DES

2. 非对称加密

  • RSA：常用非对称加密算法
  • ECC：椭圆曲线加密
  • DSA：数字签名算法

3. 哈希函数

  • SHA-256：安全哈希算法
  • MD5：已被破解，不建议使用
  • SHA-3：最新哈希标准
